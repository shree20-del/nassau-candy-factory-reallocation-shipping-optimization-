# ⚡ Nassau Candy Profitability & Margin Performance Analysis

## 📌 Overview

An interactive business intelligence dashboard built using Streamlit, Plotly, and Python to analyze product-line profitability, margin efficiency, operational cost structure, and division-level financial performance for Nassau Candy Distributor.

The project transforms raw operational and sales data into actionable business insights that support:
- profitability optimization,
- pricing decisions,
- margin risk detection,
- and portfolio rationalization.

# 🎯 Business Problem

For distributors like Nassau Candy, high sales volume does not always translate into high profitability.

Some products:
- generate strong revenue but weak margins,
- consume disproportionate operational cost,
- appear successful while reducing overall profitability.

This project helps identify:
- which products truly drive profit,
- which divisions underperform financially,
- where pricing optimization is needed,
- and which products represent structural margin risk.

# 📌 Key KPIs

| KPI | Description |
|---|---|
| Gross Margin % | Gross Profit ÷ Sales |
| Profit per Unit | Gross Profit ÷ Units |
| Revenue Contribution | Product Sales ÷ Total Sales |
| Profit Contribution | Product Profit ÷ Total Profit |
| Margin Volatility | Margin fluctuation over time |
